# Inventory Management System

A comprehensive inventory management system with multiple depots and role-based access control.

## Features

- **Multi-depot Management**: Dhaka, Chittagong, Jhenaidah, Bogra, Rangpur
- **Role-based Access Control**:
  - Admin: Full system access
  - Factory: Manage production and inventory
  - Depot: View and manage only assigned depot
- **Inventory Types**:
  - Raw Materials (KG/LTR)
  - Packing Materials (PCS)
  - Finished Goods (PCS/CARTON)
- **Production Conversion**: Convert raw materials to finished goods
- **Real-time Dashboard**: Live inventory tracking
